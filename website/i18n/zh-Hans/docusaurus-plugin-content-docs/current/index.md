# Hermes Agent

自我进化的 AI Agent，由 [Nous Research](https://nousresearch.com) 构建。它会生成和改进技能，保持跨会话记忆，并能在本地、云端或无服务器环境长期运行。

## 为什么用 Hermes？

- **长期记忆与自改进**：自动沉淀知识、生成技能、定期回顾。
- **多平台常驻**：CLI + 15+ 消息平台网关，支持语音与流式回复。
- **可移植运行环境**：本地、Docker、SSH、Daytona、Modal 等随时切换。
- **安全与可控**：命令审批、工具过滤、MCP 选择性暴露。
